Subscribe to a Nodeâ€‹

Subscribe to a Node

info

Please ensure that you select one option between <gigabytes> and <hours>. The unused option should be assigned a value of 0.

sentinelcli tx node subscribe \
  <sentnode_address> \
  <gigabytes> \
  <hours> \
  udvpn \
  --from test \
  --chain-id=sentinelhub-2 \
  --node https://rpc.sentinel.co:443 \
  --gas-prices=0.5udvpn \
  --gas=300000

Update a Node Statusâ€‹

Update the status of a node

sentinelcli tx node update-status \
  <status> \
  --from test \
  --chain-id=sentinelhub-2 \
  --node https://rpc.sentinel.co:443 \
  --gas-prices=0.5udvpn \
  --gas=300000

Update a Provider Statusâ€‹

Update the status of a provider

sentinelcli tx provider update \
  --name <new_name> \
  --identity <new_identity> \
  --website <new_website> \
  --description <new_description> \
  --status <new_status> \
  --from test \
  --chain-id=sentinelhub-2 \
  --node https://rpc.sentinel.co:443 \
  --gas-prices=0.5udvpn \
  --gas=300000

Start a Sessionâ€‹

Initiate a session on a node with an existing subscription.

sentinelcli tx session start \
  <subscription> \
  <sentnode_address> \
  --from test \
  --chain-id=sentinelhub-2 \
  --node https://rpc.sentinel.co:443 \
  --gas-prices=0.5udvpn \
  --gas=300000

  --gas-prices=0.5udvpn \
  --gas=300000
