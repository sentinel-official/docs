Sentinel Public APIs

There are currently 3 public APIs provided by Sentinel: LCD, RPC and the historical data API.

APIs

RPC endpoints
Sentinel RPC endpoints
gRPC
Sentinel high performance, universal RPC framework
LCD
State queries, legacy transactions endpoints

For a brief overview of our supply details, please visit https://api.supply.sentinel.co/

info

Please note that for maximun reliability and speed, LCD or RPC can be hosted by anyone by simple running a node and configuring the API service.
