# Import your wallet using a Private Key

To import your wallet using the private key, follow these steps:

Click on the option to Import your existing wallet “Using a private key”

![](/img/leap/import-private-key-1.png)

Paste your private key in the text box.

![](/img/leap/import-private-key-2.png)

Your Leap Wallet is ready to use now.

![](/img/leap/import-private-key-3.png)
