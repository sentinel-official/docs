## Using the HUB

Once sentinelhub is [installed](../sentinel-core/build) and configured with the correct network, you can now send commands with it. In a new terminal window, start by running the following query command:

### Node status
```bash
sentinelhub status
```

### Node configuration
```bash
sentinelhub config
```
Output:
```json
{
	"chain-id": "",
	"keyring-backend": "os",
	"output": "text",
	"node": "tcp://localhost:26657",
	"broadcast-mode": "sync"
}
```

### Change node

```
sentinelhub config --node https://rpc.sentinel.co:443
```
