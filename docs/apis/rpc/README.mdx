## Interacting with RPC endpoints

As shown on the RPC specifications, there are different endpoints to communicate with the Sentinel chain. Unlike the LCD rest api, the RPC endpoints provide generic endpoints to communicate with the various modules available. For example the [ABCI Query](/api?v=RPC#/operations/abci_query) allows you the query different data from Sentinel.

For more information please read [generating, signing and broadcasting transactions](https://docs.cosmos.network/v0.46/run-node/txs.html) on the cosmos-sdk docs.
