The query distribution module focuses on managing account balances and monitoring the coin supply.

## Query a Balance

To query an account balance, use the following command:

```bash
sentinelhub query bank balances \
  <account_address> \
  --node https://rpc.sentinel.co:443
```

## Query the Client Metadata

Query the client metadata for all the registered coin denominations

```bash
sentinelhub query bank denom-metadata \
  --node https://rpc.sentinel.co:443
```
<details>
<summary>Example Output</summary>
<p>

```bash

  denom_units:
```

```bash
sentinelhub query bank total \
  --node https://rpc.sentinel.co:443
```
