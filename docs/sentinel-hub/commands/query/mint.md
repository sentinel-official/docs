Querying commands for the minting module, which handles token inflation and block rewards.

## Query Inflation

Query the current minting inflation value.

```bash
sentinelhub query mint inflation \
  --node https://rpc.sentinel.co:443
```

## Query Annual Provisions

Query the current minting annual provisions value.

```bash
sentinelhub query mint annual-provisions \
  --node https://rpc.sentinel.co:443
```

## Query Mint Params

Query the current minting parameters.

```bash
sentinelhub query mint params \
  --node https://rpc.sentinel.co:443
```
