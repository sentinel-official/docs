# x402

x402 is a community-built payments layer that lets AI agents pay USDC on Base or Solana for private VPN access through the Sentinel dVPN network of 900+ decentralized nodes.

You can access x402 from:
- [GitHub repository](https://github.com/Sentinel-Autonomybuilder/x402)
