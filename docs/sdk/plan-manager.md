# Sentinel Plan Manager

Sentinel Plan Manager is the community-built commerce layer for Sentinel dVPN. It lets you create on-chain subscription plans, curate node pools, manage subscribers, and issue fee grants. Built on top of the community blue-js-sdk.

You can access Plan Manager from:
- [GitHub repository](https://github.com/Sentinel-Autonomybuilder/sentinel-plan-manager)
