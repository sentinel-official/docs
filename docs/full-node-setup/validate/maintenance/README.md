# Validator Maintenance

Day-to-day operations once your validator is signing blocks. Each topic is independent. Read whichever is relevant when you need it:

- **Save Your Keys!**: back up the keys you cannot afford to lose.
- **Add a Validator Image**: display a logo and metadata in explorers.
- **Useful Commands**: common day-2 ops (jailed status, voting power, querying state).
- **Server Migration**: move a validator to a new host without missing blocks.
